# Playwright Automation Demo

## Project Overview
This project demonstrates end-to-end UI automation testing using Playwright with TypeScript.

## Application Tested
https://www.saucedemo.com/

## Test Scenarios Covered
- Login validation
- Product selection
- Add to cart
- Cart verification
- Screenshot capture

## Technologies Used
- Playwright
- TypeScript
- Node.js

## Browsers Executed
- Chromium
- Firefox
- WebKit

## Execution Command

```bash
npx playwright test
```

## Generate HTML Report

```bash
npx playwright show-report
```

## Project Structure

```text
tests/
screenshots/
playwright-report/
test-results/
```

## Test Result
All tests executed successfully across supported browsers.
